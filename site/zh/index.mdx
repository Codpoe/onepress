### 类似 VuePress，容易上手

```shell
# 在你的项目中安装
yarn add -D onepress

# 新建一个 markdown 文件
echo '# Hello OnePress' > README.md

# 开始写作
yarn onepress dev

# 构建静态文件
yarn onepress build
```
